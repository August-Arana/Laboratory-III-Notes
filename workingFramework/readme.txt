# Comida rapida con sistema de control

Compilar el programa utilizando

make

Luego, ejecutar los programas obtenidos en el siguiente orden

./productor

./consumidor A
./consumidor B
./consumidor C


Luego, dentro del programa productor se mostrara las acciones realizadas por el programa, mientras se indica si cada menu querra postre o no.

Segun el tipo de menu selecionado al azar, los otros procesos mostraran el proceso ejecutado.

USO DEL PROGRAMA:

1. Primero se deber ejecutar los procesos de equipos. A cada equipo se le pasa por parametro que numero de equipo son. Se debe pasar 1 (para ser el equipo 1) o 2 (para ser el equipo 2). Se inicia un equipo, y despues el otro. Es importante inicializar los dos.
Ejemplo de ejecucion:
-Terminal 1:
./equipo 1

-Terminal 2:
./equipo 2

2.Luego se ejecuta el proceso "panel". Se debe pasar por parametro un cualquier numero entero. Este numero entero representara el maximo de goles para que un equipo gane. Por ejemplo, si se pasa el numero 3, ganara el primer equipo que llegue a tres goles.
Ejemplo de ejecuci√≥n:
./panel 3

ACLARACION IMPORTANTE: cuando se imprime por pantalla "Erro" no significa que haya ocurrido un error en el programa, sino que el equipo "erro" al arco.
